# Comandos Django

## Env

**Criar um enviroment**

>conda create --name "nome" "package"

**activar o env**

>activate "name"

**desactivar env**

>deactivate "name"

**show list of envs**

>conda env list

**To install an env package**

>conda install "name"

## Django

### Project

**create project**

>django-admin startproject "name"

### APP

**Create an django app**
>python manage.py startapp
